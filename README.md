# ideas-blog

Minimal dark Astro blog for ideas and links.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

1. Import this GitHub repo in Vercel.
2. Framework preset: Astro (auto-detected).
3. Build command: `npm run build`.
4. Output directory: `dist`.
